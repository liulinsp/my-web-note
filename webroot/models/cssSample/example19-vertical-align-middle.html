父元素大小不固定，子元素大小固定（一）
缺点：
1. 子元素大小必须固定
2. 子元素大小改变后还需要修改margin值

样式：
.parent1{
	width: 100%;
	height: 100%;
	background-color: #C7C7C7;

	position: relative;
}

.child1{
	width: 50px;
	height: 50px;
	background-color: #65FC71;

	position: absolute;
	top: 50%;
	left: 50%;
	margin-top: -25px;/* -width/2 */
	margin-left: -25px;/* -height/2 */
}

页面：
<div class="parent1">
	<div class="child1"></div>
</div>
父元素大小不固定，子元素大小固定（二）
缺点：
1. 子元素大小必须固定
2. IE6/7 不支持

原理：
absolute元素包含块的宽度 = 
	元素的left + margin-left + border-left + padding-left
	+ width + padding-right + border-right  + margin-right
	+ right
其中 left、right 默认auto, margin、padding、border默认0，
所以left、right设为0，margin设置auto就能居中了，垂直居中同理。

样式：
.parent0{
	width: 100%;
	height: 100%;
	background-color: #C7C7C7;

	position: relative;
}

.child0{
	width: 150px;
	height: 80px;
	background-color: #65FC71;
	
	position: absolute;
	top: 0;
	left: 0;
	right: 0;
	bottom: 0;
	margin: auto;
}

页面：
<div class="parent0">
	<div class="child0">
		<h2></h2>
	</div>
</div>

父子元素大小都不固定-绝对定位和位移
缺点：
1. IE8及以下不支持
2. 子元素比父元素大时会出现子元素溢出现象
样式：
.parent2{
	width: 100%;
	height: 100%;
	background-color: #C7C7C7;

	position: relative;
}

.child2{
	background-color: #65FC71;

	position: absolute;
	top: 50%;
	left: 50%;
	-webkit-transform: translate(-50%, -50%);
	-ms-transform: translate(-50%, -50%);
	transform: translate(-50%, -50%);
}	

页面：
<div class="parent2">
	<div class="child2">
		<h2>你好！</h2>
	</div>
</div>

你好！

2. 只能视口居中
代码：

<html>
<head>
	<title></title>
<style>
html,body{
	padding: 0;
	margin: 0;
	background-color: #C7C7C7;

	height:100%;
	text-align: center;	
}
.child3{
	background-color: #65FC71;
	
	display: inline-block;
	margin: 50vh auto 0;
	transform: translateY(-50%);
}	
</style>
</head>
<body>
	<div class="child3">
		<h2>你好！</h2>
	</div>
</body>
</html>
