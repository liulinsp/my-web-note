父元素大小不固定，子元素大小固定
缺点：子元素比父元素大时会出现子元素溢出现象
样式：
.parent1{
	width: 100%;
	height: 100%;
	background-color: #C7C7C7;

	position: relative;
}

.child1{
	width: 50px;
	height: 50px;
	background-color: #65FC71;

	position: absolute;
	top: 50%;
	left: 50%;
	margin-top: -25px;/* -width/2 */
	margin-left: -25px;/* -height/2 */
}

页面：
<div class="parent1">
	<div class="child1"></div>
</div>
父子元素大小都不固定-基于绝对定位
缺点：
1. IE8不支持
2. 子元素比父元素大时会出现子元素溢出现象
样式：
.parent2{
	width: 100%;
	height: 100%;
	background-color: #C7C7C7;

	position: relative;
}

.child2{
	background-color: #65FC71;

	position: absolute;
	top: 50%;
	left: 50%;
	transform: translate(-50%, -50%);
}	

页面：
<div class="parent2">
	<div class="child2">
		<h2>你好！</h2>
	</div>
</div>

你好！

父子元素大小都不固定-基于视口单位
缺点：
1. IE8不支持
2. 只能视口居中
代码：
<!DOCTYPE html>
<html>
<head>
	<title></title>
<style>
html,body{
	padding: 0;
	margin: 0;
	background-color: #C7C7C7;

	height:100%;
	text-align: center;	
}
.child3{
	background-color: #65FC71;
	
	display: inline-block;
	margin: 50vh auto 0;
	transform: translateY(-50%);
}	
</style>
</head>
<body>
	<div class="child3">
		<h2>你好！</h2>
	</div>
</body>
</html>
父子元素大小都不固定-基于弹性布局
缺点：
1. IE8不支持
样式：
.parent4{
	width: 100%;
	height: 100%;
	background-color: #C7C7C7;

	display: flex;
}

.child4{
	background-color: #65FC71;

	margin: auto;
}

页面：
<div class="parent4">
	<div class="child4">
		<h2>你好！</h2>
	</div>
</div>

你好！
