IE 滤镜
Alpha：设置透明层次. 
blur：创建高速度移动效果，即模糊效果. 
Chroma：制作专用颜色透明. 
DropShadow：创建对象的固定影子. 
FlipH：创建水平镜像图片. 
FlipV：创建垂直镜像图片. 
glow：加光辉在附近对象的边外. 
gray：把图片灰度化. 
invert：反色. 
light：创建光源在对象上. 
mask：创建透明掩膜在对象上. 
shadow：创建偏移固定影子. 
wave：波纹效果. 
Xray：使对象变的像被x光照射一样.

使用drop-shadow滤镜实现特殊形状的阴影
.speech{
	position: absolute;
	box-sizing: border-box;
	width: 300px;
	min-height: 100px;
	background-color: #FCE12B;
	border-radius: 10px;
	padding: 10px;
	line-height: 1.5;
}
.speech::before{
	position: absolute;
	content: "";
	background-color: #FCE12B;
	width: 40px;
	height: 40px;
	transform: rotate(-45deg);
	left: 0px;
	top: 20px;
	z-index: -1;
}
.shadow{
	/*for ie6,7,8*/
	filter: progid:DXImageTransform.Microsoft.Shadow(color='#969696', Direction=135, Strength=5);
	-webkit-filter: drop-shadow(.05em .05em .3em gray);
	-moz-filter: drop-shadow(.05em .05em .3em gray);
	-o-filter: drop-shadow(.05em .05em .3em gray);
	filter: drop-shadow(.05em .05em .3em gray);
}
