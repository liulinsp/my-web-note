border-image实现图片边框【IE8不支持】
样式：
.img-border{
	width: 200px;
	height: 150px;
	border-width: 10px;
	border-style: solid;/*如果没有此属性firefox不显示边框*/
	-webkit-border-image: url(../../img/dot.png)30 30 round;
	-moz-border-image: url(../../img/dot.png)30 30 round;
	-o-border-image: url(../../img/dot.png)30 30 round;
	border-image: url(../../img/dot.png)30 30 round;
}

页面：
<div class="img-border"></div>
两层背景图片方式实现图片边框【IE8不支持】
样式：
.imgbg-border{
	width: 200px;
	height: 150px;	
	border: 20px solid transparent;
	/*前面的背景图会覆盖在后面的背景图之上*/
	/*background: linear-gradient(white, white),
				url(../../img/stone-border.jpg);
	background-size: cover;
	background-clip: padding-box, border-box;
	background-origin: border-box;*/

	/*简写*/
	background:linear-gradient(white, white) padding-box,
				url(stone-art.jpg) border-box 0 / cover;		
}

页面：
<div class="imgbg-border"></div>

动态虚线边框【IE8不支持】
样式：
@keyframes ants {
	100% {background-position: 20px 20px}
}
.animation-border{
	width: 260px;
	height: 180px;
	border: 1px solid transparent;
	background: linear-gradient(#fff,#fff) padding-box,
				repeating-linear-gradient(-45deg,
					transparent 0,transparent 25%,
					#000 0,#000 50%) border-box 0 / 14.14px 14.14px;
	background-position: 0 0;
	animation: ants 1s linear infinite;
}

页面：
<div class="animation-border"></div>
