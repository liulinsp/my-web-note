/*隐藏对象。与visibility属性的hidden值不同，
	  不为被隐藏的对象保留其物理空间*/
.none-style{
	width: 100px;
	height:100px;
	display: none;
}
这个div被隐藏起来了，没有保留物理空间
上面有一个display:none的div 
/*【对比】隐藏对象。与display属性的none值不同，
	  保留隐藏的对象的物理空间*/
.visibility-style{
	width: 100px;
	height:100px;
	visibility: hidden;
}
这个div被隐藏起来了，保留了物理空间
上面有一个visibility: hidden的div 
/*默认。此元素会被显示为内联元素，元素前后没有换行符*/
.inline-style{
	width: 100px;
	height:100px;
	background-color: #F9F9CF;
	display: inline;
}
这个div是内联元素
上面有一个display: inline的div 
/*此元素将显示为块级元素，此元素前后会带有换行符*/
.block-style{
	width: 100px;
	height:100px;
	background-color: #F9F9CF;
	display: block;
}
这个span是块元素 上面有一个display: block的span 
/*行内块元素。（CSS2.1 新增的值）*/
.inline-block-style{
	width: 100px;
	height:100px;
	background-color: #F9F9CF;
	display: inline-block;
}
这个span是行内块元素 上面有一个display: inline-block的span 
/*此元素会作为列表显示*/
.list-item-style{
	width: 200px;
	height: 50px;
	background-color: #F9F9CF;
	display: list-item;
}
这个span是列表元素
这个div是列表元素
上面有一个display: list-item的span和一个display: list-item的div 
/*此元素会作为块级表格来显示（类似 table标签），
  表格前后带有换行符*/
.table-style{
	width: 200px;
	height: 50px;
	background-color: #F9F9CF;
	display: table;
}
这个div是表格元素
上面有一个display: table的div 
/*此元素会作为内联表格来显示（类似 table标签），
  表格前后没有换行符*/
.inline-table-style{
	width: 200px;
	height: 50px;
	background-color: #F9F9CF;
	display: inline-table;
}
这个div是内联表格元素
上面有一个display: inline-table的div
