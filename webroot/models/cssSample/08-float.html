如果float不是none，当display:inline-table时，display的计算值为table；当display:inline | inline-block | run-in | table-* 系时，display的计算值为block，其它情况为指定值；

float在绝对定位和display为none时不生效。

对应的脚本特性为styleFloat（IE）或cssFloat（非IE）。(注意这里为styleFloat或cssFloat，而不是float)

.float-none{
	float: none;
	display: inline-block;

对于CSS的清除浮动(clear)，
只能影响使用清除的元素本身，不能影响其他元素
/*允许两边都可以有浮动对象*/
.clear-none{
	clear:none;
}
/*不允许左边有浮动对象*/
.clear-left{
	clear:left;
}

/*不允许右边有浮动对象*/
.clear-right{
	clear:right;
}

/*不允许有浮动对象*/
.clear-both{
	clear:both;
}

上例最后一个浮动块太多导致溢出父div,
这时父div需要加入新的样式：
.clearfix {
  overflow: auto;
}
如果你想要支持IE6，你就需要再加入如下样式：
.clearfix {
  overflow: auto;
  zoom: 1;
}
