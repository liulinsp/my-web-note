media type 媒体类型

	
		
值描述

	
	
		
all用于所有设备

		
screen用于电脑屏幕，平板电脑，智能手机等

		
print用于打印机和打印预览

		
speech应用于屏幕阅读器等发声设备
media feature 媒体特征

	
		
特征可加min-/max-前缀描述

	
	
		
width是页面可见区域宽度，示例：max-width: 1024px

		
height是页面可见区域高度

		
device-width是屏幕宽度

		
device-height是屏幕高度

		
orientation否屏幕方向，portrait：竖屏，landscape：横屏

		
aspect-ratio是width 与 height 的比值,示例：aspect-ratio:1680/957

		
device-aspect-ratio是device-width 与 device-height 的比值

		
color是设备每一组彩色原件的个数。如果不是彩色设备，则值等于0，示例：max-color: 8

		
color-index是设备的彩色查询表中的条目数。如果没有使用彩色查询表，则值等于0

		
monochrome是在单色缓冲区中，每像素包含的单色原件个数。如果不是单色设备，则值等于0

		
resolution是设备像素密度, 示例：max-resolution: 96dpi

		
scan否电视类设备的扫描方式，progressive:逐行扫描，interlace:隔行扫描(？？？)

		
grid否设备是否使用网格，1: 基于网格的设备，0：非网格设备（例：电脑显示器）
语法
CSS语法：
@media mediatype and|not|only (media feature) {
    CSS-Code;
}

针对不同的媒体使用外部样式表:
<link rel="stylesheet" media="mediatype and|not|only (media feature)" href="mystylesheet.css">

only的用法：
<link rel="stylesheet" media="only screen and (max-device-width:240px)" href="mystylesheet.css">
* only用来排除不支持媒体查询的浏览器
* 其实only很多时候是用来对那些不支持Media Query但却支持Media Type的设备隐藏样式表的
* 支持媒体特性（Media Queries）的设备，正常调用样式，此时就当only不存在；
* 对于不支持媒体特性(Media Queries)但又支持媒体类型(Media Type)的设备，这样就不会读样式了

媒体查询实现自适应栅格布局
IE8及以下不支持
CSS 语法
@media mediatype and|not|only (media feature) {
    CSS-Code;
}
样式：
.content{
	background-color: #DFFBED;
	word-spacing: -6px;
	overflow: auto;
}
.col-item{
	float: left;
  	min-height: 1px;
	width: 20%;
	box-sizing:border-box;
	border: 1px solid #000; 
	background-color: #FCE2FA;
}
@media screen and (max-width: 1024px) {
	.col-item{
		width: 25%;
	}
}
@media screen and (max-width: 680px) {
	.col-item{
		width: 50%;
	}
}
@media screen and (max-width: 320px) {
	.col-item{
		width: 100%;
	}
}

页面：
<div class="content">
	<div class="col-item">1</div>
	<div class="col-item">2</div>
	<div class="col-item">3</div>
	<div class="col-item">4</div>
	<div class="col-item">5</div>
</div>
