.css( propertyName ) 
获取匹配元素集合中的第一个元素的样式属性的值
propertyName 类型: String,一个css属性名。

.class1{

.class1{

.css( propertyName, value )
.css( propertyName, value ) 
设置每个匹配元素的CSS属性。
propertyName
类型: String
一个CSS属性名
value
类型: String, Number
设置这个CSS属性的值

样式：
div.class2{
  display: inline-block;
  background-color: #D9D8FC;
  width: 30px;
  height: 20px;
}

脚本：
$(".class2").css("border","solid 1px red");
.css( properties )
.css( properties )
properties 类型: PlainObject
一个 属性-值 配对的对象
样式：
div.class3{
  display: inline-block;
  background-color: #FCD8F8;
  width: 40px;
  height: 20px;
}
脚本：
$(".class3").css({
  'border-style': 'solid',
  'border-color': '#FC88FD',
  'border-width': '2px'
});
.css( propertyName, function(index, value) )
.css( propertyName, function(index, value) )
propertyName
类型: String
一个CSS属性名

function(index, value)
类型: Function()
一个返回设置值的函数。this 是当前元素。
接收index 参数表示元素在匹配集合中的索引位置
和html 参数表示元素上原来的 HTML 内容。
样式：
div.class4{
  display: inline-block;
  background-color: #CA486D;
  width: 50px;
  height: 20px;
}
脚本：
$(".aShow>div").css("border-radius",function(index,value){
  if($(this).hasClass('class4')){
    return "10px";
  }else{
    return "0px";
  }
});
