.val()
.val()方法主要用于获取表单元素的值，比如 input, select 和 textarea。
对于 <select multiple="multiple"> 元素, .val()
方法返回一个包含每个选择项的数组，如果没有选择性被选中，它返回null。

<input type="text" name="username" value="Tom">
<input type="password" name="psw" value="123456">

<input type="radio" name="sex" value="man">男
<input type="radio" name="sex" value="woman" checked="checked">女

<input type="checkbox" name="modules" value="ctrl" 
  checked="checked">控制模块
<input type="checkbox" name="modules" value="proxy" 
  checked="checked">代理模块

<select name="province">
<option value="11">北京</option>
<option value="12" selected="selected">天津</option>

</select>

.val( function(index, value) )
.val( function(index, value) )
function(index, value) 类型: Function()
一个用来返回设置值的函数。this 指向当前元素。
接收的集合中的元素，旧的值作为参数的索引位置。

<input type="text" name="loginname" value="superman">
