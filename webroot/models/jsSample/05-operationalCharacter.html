逗号运算符
逗号操作符 对它的每个操作对象求值（从左至右），然后返回最后一个操作对象的值。

//1.逗号运算符常用于声明多个变量
var iNum1 = 1, iNum = 2, iNum3 = 3;

//2.逗号运算符在for循环中
for(var i=0,j=10 ; i < j ; i++,j--){
	
}

//3.逗号运算符还可以用于赋值
var num = (1,2,3,4,5);//num值为5

//4.例题
var i, j, k;
for (i=0, j=0; i < 10, j < 6; i++, j++) {
  k = i+j;
}
最后k的值为10

//5.运用逗号运算符交换变量
var a = "a", b = "b";
a = [b][(b = a, 0)];
//a = [b, b = a][0];

//6. 简化代码
if(x){
  foo();
  return bar();
}
else{
  return 1;
}

简化为：return x ? (foo(), bar()) : 1;
void运算符
void运算符的作用是:计算表达式expr,并返回undefined。

1. 链接中用void(0)代替undefined

2.IE中返回值问题
javascript:window.open("http://www.whitehouse.gov/") 
会替换掉当前页面的内容,显示[object]

javascript:void window.open("http://www.whitehouse.gov/") 
不会改变当前页面的内容.
href="javascript:undefined" 不跳转
href="javascript:void 0" 不跳转
