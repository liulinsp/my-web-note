逻辑与&&
1. 两个操作数都是布尔值时：
   只有两个值都为true才返回true,
   其他情况返回false.
2. 有一个操作数不是布尔值时（此逻辑对第一条同样适用）：
   如果第一个操作数转换为布尔值后的结果为false,
   返回第一个操作数，
   否则返回第二个操作数
逻辑或||
1. 两个操作数都是布尔值时：
   只有两个值都为false才返回false,
   其他情况返回true.
2. 有一个操作数不是布尔值时（此逻辑对第一条同样适用）：
   如果第一个操作数转换为布尔值后的结果为true,
   返回第一个操作数，
   否则返回第二个操作数
逻辑与&&应用
判断对象是否存在，如果存在再执行某些操作：
var myConsole = {
	log:function(msg){
		$('
'+msg+'
').appendTo($("#showResult5"));
	}
}
window.myConsole&&myConsole.log("myConsole存在");
window.noConsole&&noConsole.log("noConsole不存在");
//直接用noConsole会报错，Uncaught ReferenceError: noConsole is not defined
//noConsole.log("noConsole不存在");

最常用的是IE8中没有window.console对象，为避免报错：
window.console&&console.log("test");
逻辑或||应用
自定义的配置与默认配置合并：
    1. 默认提示信息defaultMsg="保存失败！";
    2. 如果参数传了自定义的提示信息msg="用户信息保存失败！"，
       则显示自定义的提示信息;
    3. 否则显示默认提示信息。

var defaultMsg="保存失败！";

function showMsg(msg){
	var message = msg||defaultMsg;
	$('<p>'+message+'</p>').appendTo($("#showResult3"));
}

showMsg();
showMsg("用户信息保存失败！");
综合应用
等级划分：
根据学生考试分数划分等级
1. >=90 , A级
2. >=80 并且 < 90, B级
3. >=70 并且 < 80, C级
4. >=60 并且 < 70, D级
4. < 60 , E级

function getLevel(score){
	if(isNaN(score)){
		return "";
	}
	return (score>=90&&"A")||(score>=80&&"B")||
		   (score>=70&&"C")||(score>=60&&"D")||"E";
}
