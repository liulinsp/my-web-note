概念
Web Storage 包含如下两种机制：sessionStorage和localStorage
用来做本地存储，浏览器为每个站点提供了独立的存储区域(5M)
1. sessionStorage 
   存储区域只在页面会话期间可用，浏览器关闭数据就无效了
2. localStorage 
   在浏览器关闭，然后重新打开后数据仍然存在。

兼容性：
1. IE6、IE7不支持
2. 从ios 5.1后，Safari移动存储本地存储的数据在缓存文件夹，
	这样在系统空间不足的情况下，方便系统自动清理。
localStorage操作
1. 设置
window.localStorage.setItem('sessionId','12345')

2. 读取
var sessionId = window.localStorage.getItem('sessionId')

3. 遍历
var data = {};
for(var i=0; i < window.localStorage.length; i++){
	var key = window.localStorage.key(i);
	data[key] = window.localStorage.getItem(key);
}
