转字符串
1.调用被转换类型的toString()方法
例子：
	var a = 123;
	a = a.toString();
- 注意，该方式不适用于null和undefined，如果对这两个调用toSting()浏览器会报错

2.调用String()函数
例子：
	var b = true;
	b = String(b);
- 原理：对于有toString()方法它会调用toString()
		而对于null，它会直接转换为"null"
		对于undefined，它会直接转换为"undefined"

3.使用 +"", 进行隐式类型转换
例子：
	var c = 567;
	c = c+"";
- 原理：结果和String()函数一样，只不过它由浏览器底层调用，使用起来更加的简单
转数字
1.Number()函数
	- 例子：
		var a = "123";
		a = Number(a);
		- 转换的情况
			字符串：
				- 如果字符串是一个合法数字，则直接转换为对应的数字
				- 如果是一个非法数字，则直接转换为NaN
				- 如果是一个空串，或者都是空格则会转换为0
			布尔值：
				- true --> 1
				- false ---> 0
			其他：
				- null --> 0
				- undefined --> NaN

2.parseInt()和parseFloat()
	- 这两种方式专门用来将一个字符串转换为一个Number
	- parseInt()可以将一个字符串中的有效的整数位转换为Number
	- parseFloat()可以将一个字符串中的有效的小数转换为Number

3.使用一元加运算符（+）, 进行隐式类型转换
	- 例：
	var a = "123";
	a = +a;
	- 结果和使用Number()函数一样
转布尔值
1.使用Boolean函数
- 例子：
	var a = 1;
	a = Boolean(a);
- 不同的情况
	字符串：
		- 除了空串都是true
	数值
		- 除了0和NaN都是true
		
	null和undefined都是false	
	
	非null对象都是true
2.使用非运算符（!!）, 进行隐式类型转换
- 例子：
	var a = 1;
	a = !!a;
- 结果和使用Boolean()函数一样
