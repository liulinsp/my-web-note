namespace模式
namespace模式: 简单对象封装
作用: 减少了全局变量
问题: 不安全(数据不是私有的, 外部可以直接修改)

页面引用：
<script type="text/javascript" src="module1.js"></script>

module1.js
var myModule = {
  data: '我是myModule的属性',
  foo() {
    console.log('foo() '+this.data)
  },
  bar() {
    console.log('bar() '+this.data)
  }
}

脚本：
  myModule.foo()
  myModule.bar()
  function foo() {
    console.log('foo() other data')
  }
  myModule.foo()

  myModule.data = 'xxxx'
  myModule.foo()
