网格
在学习了几何形状和材质之后，我们就能使用他们来创建物体了。
最常用的一种物体就是网格（Mesh），网格是由顶点、边、面等组成的物体；
其他物体包括线段（Line）、骨骼（Bone）、粒子系统（ParticleSystem）等。
位置
// 球体
var cube = new THREE.Mesh(
	new THREE.SphereGeometry(1, 64, 64),
    material
);
// cube.position.z = -1;
cube.position.set(-1, -1, -2);
// cube.position = new THREE.Vector3(-1, -1, -2); // 无效？？？
scene.add(cube);
缩放
cube.scale.set(2, 1.5, 1);
旋转
cube.rotation.set(Math.PI / 4, Math.PI / 4 , 0);
