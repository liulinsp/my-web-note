AngularJS简介
AngularJS 通过 ng-directives 扩展了 HTML。
ng-app 指令定义一个 AngularJS 应用程序。
ng-model 指令把元素值（比如输入域的值）绑定到应用程序。
ng-bind 指令把应用程序数据绑定到 HTML 视图。
ng-init 指令初始化 AngularJS 应用程序变量

当网页加载完毕，AngularJS 自动开启。
ng-app 指令告诉 AngularJS，<div> 元素是 AngularJS 应用程序 的"所有者"。
ng-model 指令把输入域的值绑定到应用程序变量 name。
ng-bind 指令把应用程序变量 name 绑定到某个段落的 innerHTML。
model
<div ng-app="">
	 <p>名字 : <input type="text" ng-model="name"></p>
	 <h1>Hello {{name}}</h1>
</div>

名字 :

Hello {{name}}

init和bind
<div ng-app="" ng-init="firstName='张';lastName='亮'">
	<span ng-bind="firstName" style="color:#261A7D"></span>
	<span ng-bind="lastName" style="color:#187F20"></span>
	<h2>{{firstName + "·" + lastName}}</h2>
</div>

{{firstName + "·" + lastName}}

扩展属性
HTML5 允许扩展的（自制的）属性，以 data- 开头。
AngularJS 属性以 ng- 开头，但是您可以使用 data-ng- 来让网页对 HTML5 有效。

<div data-ng-app="" data-ng-init="name='张三'">
	姓名：<span data-ng-bind="name"></span>
</div>
姓名：
控制器
AngularJS 模块（Module） 定义了 AngularJS 应用。
AngularJS 控制器（Controller） 用于控制 AngularJS 应用。
ng-app指令定义了应用, ng-controller 定义了控制器。

<div ng-app="app1" ng-controller="ctrl1">
	姓名：<input ng-model="name">
</div>

脚本：
var app1 = angular.module("app1",[]);
app1.controller("ctrl1",function($scope){
	$scope.name="张三";
});
